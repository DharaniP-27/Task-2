🔐 Vulnerability Scanner (Mini Project)

A simple Python-based Vulnerability Scanner that helps identify common security weaknesses in websites and networks. This project scans for open ports, checks HTTP security headers, detects potential misconfigurations, and generates a vulnerability assessment report.

🚀 Features

* Scan commonly used network ports
* Detect open services running on a target system
* Check important HTTP security headers
* Identify weak or missing security configurations
* Generate a vulnerability scan report
* Beginner-friendly cybersecurity project

🛠️ Technologies Used

* Python 3
* Socket Programming
* Requests Library

📋 How It Works

1. Enter the target IP address.
2. Enter the target website URL.
3. The scanner checks for open ports.
4. Security headers are analyzed.
5. A report is generated with the findings.
